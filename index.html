업무 수정·삭제

각 업무 카드 오른쪽 버튼:

  • ✏️ (연필): 수정 — 기존 내용이 채워진 상태로 모달 열림
  • 🗑️ (휴지통): 삭제 — 확인 후 영구 삭제 (되돌릴 수 없음)

검색·필터

  • 검색창: 담당자 이름 또는 업무 내용으로 검색
  • 필터 버튼: 전체 / 예정 / 진행중 / 완료 / 보류 선택
완료 상태 업무는 완료 처리 후 7일간만 보드에 표시됩니다.
오래된 완료 업무는 보고서 탭에서 기간 검색으로 조회하세요.

📁 프로젝트별 탭

업무를 카테고리(프로젝트) 기준으로 묶어서 보여줍니다.

  • 각 카테고리 안에서 예정/진행중/완료/보류 개수를 한눈에 확인
  • 카테고리 헤더를 클릭하면 접고 펼칠 수 있음
업무 추가 시 프로젝트/카테고리 항목을 입력해야 이 탭에서 분류됩니다.

📊 SNS 수치 탭

팀의 SNS 채널별 팔로워·구독자 수를 매일 기록하는 곳입니다.

수치 입력 방법

  1. 원하는 채널 카드 하단 입력창에 숫자 입력
  2. 저장 버튼 클릭
  3. 전날 대비 증감(▲▼)이 자동으로 표시됨
  4. 최근 14일치 추이 그래프가 자동으로 그려짐

지원 채널: YouTube / FB 페이지 / FB 그룹 / Instagram / Twitter/X / Threads

🔄 파이프라인 탭

콘텐츠 제작 단계를 칸반 보드 형식으로 시각화합니다.

단계 순서: 예정 → 기획 → 진행 중 → 제작완료 → 검수 → 게시

  • 업무 카드의 ◀ ▶ 화살표로 단계 이동
  • 상단 채널 필터로 특정 SNS 채널만 보기 가능
파이프라인에 표시되려면 업무 추가 시 SNS 채널을 반드시 선택해야 합니다.

자주 묻는 질문

Q. 사이트가 갑자기 느리거나 데이터가 안 불러와져요.

Supabase 무료 플랜은 7일간 접속이 없으면 서버가 자동으로 잠듭니다. 처음 접속 시 30초~1분 기다리면 깨어납니다.

Q. 완료 처리한 업무가 보드에서 사라졌어요.

완료 상태 업무는 완료 처리 후 7일이 지나면 보드에서 자동으로 숨겨집니다. 보고서 탭에서 기간 검색으로 조회하세요.

Q. 파이프라인 탭에 업무가 안 보여요.

업무 추가 시 SNS 채널을 선택한 업무만 파이프라인에 표시됩니다. 수정 버튼으로 채널을 추가해주세요.

Q. 이미지를 첨부했는데 안 보여요.

이미지는 Supabase Storage에 업로드됩니다. work-images 버킷이 Public으로 설정되어 있어야 합니다.

Q. 한 사람이 수정하면 다른 사람 화면에도 바로 반영되나요?

네. 실시간 동기화 기능이 있어 팀원 여러 명이 동시에 접속해도 즉시 반영됩니다. 새로고침 불필요.
