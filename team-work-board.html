담당자 필터

탭 상단에 이름 칩이 표시됩니다. 클릭하면 해당 사람 업무만 보입니다. 전체를 누르면 다시 전부 표시.

📊 SNS 수치 탭

유튜브, 페이스북, 인스타그램 등 채널별 구독자/팔로워 수를 날짜별로 기록하고 추이를 확인하는 탭입니다.

수치 입력하는 법

  1. 채널 카드 하단의 날짜를 선택합니다 (기본값: 오늘)
  2. 숫자 입력란에 수치를 입력합니다
  3. 저장 버튼을 클릭합니다

저장하면 그래프에 자동 반영되고 전일 대비 증감(▲▼)이 표시됩니다.

출력하는 법

오른쪽 상단 출력 버튼 클릭 → A4 한 장짜리 채널 현황 보고서 인쇄

📋 현황 탭

전체 업무 현황을 한눈에 보는 대시보드입니다.

필터 사용법

필터선택지설명
기간전체 / 이번 주 / 이번 달해당 기간 업무만 표시
유형전체 / 쇼츠 / 카드뉴스 / 행정해당 유형만 표시
담당자이름 칩 클릭해당 사람 업무만 표시

전체 보기

유형 필터에서 전체를 선택하면 쇼츠·카드뉴스·행정이 색상 구분 박스로 분리되어 표시됩니다. 각 박스 안에 단계별 건수, 진행률 바, 담당자별 테이블이 나옵니다.

인쇄하는 법

오른쪽 상단 🖨️ 인쇄 버튼 클릭 → 현황 대시보드만 깔끔하게 출력됩니다.

자주 묻는 질문

Q. 사이트가 갑자기 느리거나 데이터가 안 불러와져요.

서버가 일정 시간 미접속 시 잠시 절전 상태가 됩니다. 처음 접속 시 30초~1분 기다리면 정상 작동합니다.

Q. 업무를 추가했는데 칸반에 안 보여요.

업무 유형과 탭이 맞는지 확인하세요. 쇼츠로 추가한 업무는 🎬 쇼츠 탭에만, 카드뉴스는 🃏 카드뉴스 탭에만 표시됩니다.

Q. 한 사람이 수정하면 다른 사람 화면에도 바로 반영되나요?

네. 실시간 동기화 기능이 있어 팀원 여러 명이 동시에 접속해도 즉시 반영됩니다. 새로고침 불필요.

Q. 삭제한 업무를 되돌릴 수 있나요?

삭제는 영구적으로 복구가 불가능합니다. 삭제 전 반드시 확인하세요.

Q. 모니터링 탭에 영상이 안 떠요.

YouTube API 키가 필요합니다. 탭 상단 🔑 API 설정에서 키를 입력하면 실제 영상이 불러와집니다. 키가 없으면 샘플 데이터가 표시됩니다.
